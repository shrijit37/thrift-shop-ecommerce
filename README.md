
# Thrift shop 

Welcome to our thrift shop! 🛍️ Feel free to browse around and find some hidden gems. 😊 We have a little bit of everything, so take your time and enjoy the hunt for that perfect find. Happy shopping! 🛒💫



## Features

- Responsive web application
- Admin Page


## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express







## Run Locally

Clone the project

```bash
  git clone https://github.com/shrijit37/thrift-shop-ecommerce.git
```

Go to the project directory

```bash
  cd moviDB
```

Install dependencies

```bash
  npm install
```

### Start the server
#### To run frontend 

```bash
npm run frontend
```

#### To run backend 

```javascript
npm run backend
```
#### To run full stack 

```javascript
npm run fullstack
```


## files not include

- .env
- /node_modules


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT NUMBER`

`MONGODB_URI`
